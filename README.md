# Weather-App

A simple weather app website that allows the user to enter a city or country and the weather will be displayed.
There is also a description of what the temp actually feels like along with a humidity %


# TECH

Used JS HTML & CSS 
The API is from the open weather map API
I used a dotenv plug in to secure the API key
